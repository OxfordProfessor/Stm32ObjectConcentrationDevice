## [基于树莓派OPENCV库与STM32F407的云台色彩跟踪系统](https://github.com/OxfordProfessor/Stm32ObjectConcentrationDevice)
作者：张凯洋

本程序是基于STM32F407芯片FreeRTOS操作系统设计的云台色彩追踪系统，采用树莓派，外置摄像头，视觉处理部分采用OPENCV库处理。
通过树莓派外置USB3.0串口通信到STM32，来操作云台追踪物体。


    版本：V1.1
        1. 可以在显示屏上显示基本的坐标信息
        2. 两轴云台可以实现操控
        3. 串口可以正常接收到数码派传回的数据
